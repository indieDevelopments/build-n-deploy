# Docker setup

This guide describes pure docker installation on `ubuntu/trusty64` Vagrant's box

## Prerequisites
- 64-bit Linux distributive
- 3.10+ kernel version

## How to
```
$ chmod +x docker-setup.sh
$ ./docker-setup.sh
```
