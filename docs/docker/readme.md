# Docker install

This guide describes pure docker installation on `ubuntu/trusty64` Vagrant's box.

## Prerequisites
- 64-bit Linux distributive
- 3.10+ kernel version
- [docker-install.sh](../scripts/docker-install.sh)

## How to
```
$ chmod +x docker-install.sh && ./docker-install.sh
```
